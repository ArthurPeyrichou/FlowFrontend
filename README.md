Configuration
=============

**En mode développement :**

Pour démarrer le serveur vue.JS, aller dans le dossier /vuejs_ptoject et taper :

`npm install`
`npm run serve`

> Le serveur tourne sur le port 4200 par default (http://localhost:4200)
